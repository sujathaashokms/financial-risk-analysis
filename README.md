# financial-risk-analysis
Credit Risk and Market Risk Analysis using Machine Learning and Python
# Financial Risk Analytics Project

This project contains two analyses:

1. Credit Risk Analysis
2. Market Risk Analysis of Indian Stocks

## Tools Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Machine Learning Models
- Logistic Regression
- Random Forest

## Key Techniques
- Exploratory Data Analysis (EDA)
- KNN Imputation
- StandardScaler
- SMOTE
- VIF for multicollinearity

## Results
The Logistic Regression model performed better with a recall of ~30% for defaulters.
